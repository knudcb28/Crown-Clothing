
# Crown Clothing

Crown Clothing an e-commerce app that uses React.js and Google Firebase for authentication. 

## Installation

Clone this project

```bash
git clone https://github.com/knudcb28/Crown-Clothing.git
```

Use `npm` to install dependencies

```bash
npm install 
```

## Usage

```Local Host
npm start
```
